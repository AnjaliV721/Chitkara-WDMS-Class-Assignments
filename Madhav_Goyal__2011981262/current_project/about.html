About This Website

This website can manipulate text in various ways you may require, like converting it to lower case, UPPER CASE, Title Case, camel Case, or removing extra spaces or even removing all spaces should you need it.

The website was built by Madhav Goyal (GitHub Profile) as his WDMS Express assignment.

The website is made using Bootstrap as the Front End Framework, Node.js as the JavaScript runtime environment, and Express.js as the Back End Framework.
